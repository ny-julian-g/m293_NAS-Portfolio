Installation von Ubuntu Server

Ein schlankes und stabiles Betriebssystem bildet die Grundlage für alle NAS-Funktionen.

Ubuntu Server wurde als Betriebssystem gewählt, da es stabil, weit verbreitet und gut dokumentiert ist. Die Installation erfolgt über ein bootfähiges USB-Medium. Wichtige Schritte: ISO herunterladen Bootfähigen USB erstellen Installation starten Benutzer und SSH einrichten Während der Installation wurde bewusst auf eine grafische Oberfläche verzichtet, um Ressourcen zu sparen. Die Verwaltung erfolgt später über SSH oder Webinterfaces.
