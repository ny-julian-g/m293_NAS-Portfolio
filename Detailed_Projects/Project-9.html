Docker und Dienste

Streame Filme und Serien direkt von deinem NAS, oder Hoste dein eigenes WordPress mit Docker.

Ein Media Server organisiert und streamt Medien auf verschiedene Geräte. Mit GPU-Unterstützung (RTX 3060) ist Hardware-Transcoding möglich. Funktionen: automatische Metadaten Streaming auf TV, Smartphone Benutzerprofile. Docker ermöglicht das einfache Bereitstellen von Diensten wie WordPress, Nextcloud etc. Containerisierung sorgt für einfache Updates und Portabilität.
